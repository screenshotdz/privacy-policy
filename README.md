## 🎨 Customization

### Modifying Colors

Edit the CSS variables in the `<style>` section:

```css
:root {
  --accent: #2563eb;        /* Main brand color (light mode) */
  --bg: #f5f7fb;            /* Background (light mode) */
  --surface: #ffffff;       /* Card/surface color (light mode) */
  --text: #1f2937;          /* Text color (light mode) */
}

@media (prefers-color-scheme: dark) {
  :root {
    --accent: #60a5fa;      /* Main brand color (dark mode) */
    --bg: #0f172a;          /* Background (dark mode) */
    /* ... other dark mode variables ... */
  }
}
```

### Adjusting Typography

Modify font sizes and line-height in the `body` CSS rule:

```css
body {
  font-family: "Your Font", sans-serif;
  line-height: 1.7;         /* Adjust spacing between lines */
  font-size: 1rem;          /* Base font size */
}
```

### Changing Layout Width

Edit the `.container` max-width:

```css
.container {
  max-width: 900px;  /* Increase/decrease for wider/narrower layout */
}
```
